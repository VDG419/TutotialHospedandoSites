Passos para hospedar um projeto no Github pages

    1. Crie um repositorio no GitHub com o mesmo nome de seu projeto
    2. No terminal, inicialize o repositorio com git init
    3. ALtere o nome da branch principal para main
    4. Adicione arquivos com git add. e faça o commit com git commit -m "first commit"
    5. Conecte ao repositorio remoto com
      git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
    6. Envie os arquivos com git push -u origin main
    7. No GitHub, vá Settings > Pages
    8. EM "Branch", selecione main e a pasta / (root) . Clique em Save
    9. Após alguns segundos, seu site estará disponivel no link exibido em "pages"
